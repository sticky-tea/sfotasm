# sfotasm: NES assembler (6502)
Assembler for 6502 NES/Dendy assembly.

## How to install
```bash
$ cd sfotasm && make && sudo make install
```

## How to use
```bash
$ sfotasm inputfile.asm [outputfile.nes]
```
